## Additional changes: 2026-04-16

### refactor: split sim config, move calibration tools

- Created `fpwfsc/sim/` with `sim_config.ini` + `sim_config.spec` extracted
  from `san/sn_config.ini`. Non-san consumers (qacits, satellite, calibration
  GUI) now read sim params from `sim/` instead of `san/`.
- Moved `detector_calibration_GUI.py` and `detector_calibration_script.py`
  from `san/` to `fpwfsc/calibration/`.
- `pyproject.toml`: `include = ["fpwfsc", "fpwfsc.*"]` so new subpackages
  are installed.
- Fixed satellite `run_satellite.py` which was reading
  `hwsettings['CAMERA_CALIBRATION']` from the sim config — that section
  never belonged there. Now defaults to None calibrations.

### equalize_image: border-median background estimation

Replaced the scalar `np.median(data)` fallback with
`estimate_background_from_border(data)` — computes column-by-column then
row-by-row medians from the outer 5 rows/columns. Captures channel biases
and row gradients that IR detectors (NIRC2) produce. Same speed as the
scalar median on 128x128 images.

### FakeDetector: deterministic bad pixels

Replaced `create_bad_pixel_mask` (random, non-reproducible) with
`deterministic_bad_pixels` (Murmur3-style bit-mixing hash on pixel
coordinates). Same pixels flagged regardless of image size, no stored
mask file needed. Config param renamed `bad_pixel_mask` →
`bad_pixel_fraction` (float 0-1). Both `sim_config` and `sn_config`
updated.

### center_image: fix coordinate swap + add clipping

The clipping rewrite introduced a coordinate swap that placed PSFs at
transposed positions. Fixed by matching the original's `[row:, col:]`
indexing. Also clips gracefully when the small image extends past the
canvas boundary (previously crashed).

### sim_image_generator: combined tool

Merged `qacits/qacits_camera_image_generator.py` and
`sim/sim_camera_image_generator.py` into `sim/sim_image_generator.py`.
Two modes: `--show` (diagnostic display) or `<dir>` (FITS stream for
hitchhiker testing). Uses `offset_tiptilt` for initial offset and drift.

### Plotter improvements

- qacits plotter: pixel value readout on hover (x, y, val label).
  Plotter stays open on Stop (shows last frame). Window taller (300x450).
- fnf plotter: pixel value readout on hover for PSF and wavefront panels.
  PSF shows raw hcipy intensity values; WF shows masked wavefront phase.
